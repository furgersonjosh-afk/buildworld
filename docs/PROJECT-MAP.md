## Folder structure

```
buildworld/
├── index.html              ← the whole app today (HTML + CSS + JS in one file)
├── manifest.json           ← makes it an installable app (PWA)
├── sw.js                   ← service worker: offline + install (bump CACHE on lib changes)
├── icon-192/512.png,       ← app icons (off-road truck). Regenerated by scripts/genicons.js
│   apple-touch-icon.png
├── lib/                    ← 3D engine, vendored LOCALLY so it works offline
│   ├── three.min.js        ← three.js r137
│   ├── OrbitControls.js    ← drag-to-spin
│   ├── RoundedBoxGeometry.js
│   ├── RoomEnvironment.js  ← instant reflections (fallback)
│   ├── RGBELoader.js  +  studio.hdr   ← real studio lighting
│   └── GLTFLoader.js        ← loads real .glb car models
├── assets/
│   ├── models/             ← DROP REAL 3D CAR MODELS (.glb) HERE  (see its README)
│   └── textures/           ← future: floors/materials (for the house app too)
├── src/                    ← reserved: when index.html gets split into modules
└── docs/
    ├── PROJECT-MAP.md      ← this file
    └── BACKLOG.md          ← the work, split into jobs/phases
```

> Note: today all the code is in `index.html` on purpose (zero build step, opens
> anywhere). When it grows we split logic into `src/*.js` (loaded via `<script>`
> tags, no bundler — three.js stays global).

## How to extend (add content = add a row)

- **New car body:** add to `VEHICLES`; add a `CAR_PROFILE` entry for its 3D shape;
  add a `carSpec3D` case; (optional) add `carDims` for Test Drive.
- **Real/realistic car:** drop a `.glb` in `assets/models/`, then add it to
  `CAR_MODELS` (e.g. `supercar: "assets/models/apex-gt.glb"`). Done.
- **New engine / part / building / job:** add a row to `ENGINES` / `PARTS` /
  `BUILD_ITEMS` / `JOBS`.

## Deploy pipeline

Push to `master` → GitHub Pages rebuilds automatically (~1–2 min) → live site
updates. The service worker is network-first for the page, so players get updates
when online. Bump `CACHE` in `sw.js` when you change files in `lib/`.

To work on it: clone, edit, `git push`. (Icons: `node scripts/genicons.js .`)
